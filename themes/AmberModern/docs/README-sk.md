# 🎨 Farebná téma Amber Modern

> Pre každého, koho unavuje klasický modrý vzhľad Far Managera a chce modernú, pokojnú a príjemnú farebnú schému.

![Celkový vzhľad](demo.png)

## 📓 Poznámky

1. Téma nemení pravidlá farebného zvýraznenia súborov — používajú sa pravidlá už nastavené vo Far Manageri.
2. Téma nastavuje základné farby editora a prehliadača, ale neobsahuje vlastné zvýraznenie syntaxe. Pre editor používam [Monokai](https://github.com/joric/monokai) a odporúčam ho vyskúšať.

## 📦 Inštalácia

Amber Modern používa rozšírenú paletu farieb, preto vyžaduje zapnuté **Virtual Terminal Rendering**. Potrebné nastavenie je už zapnuté v súbore témy.

1. [Stiahnite tému](https://github.com/LanKing/far-tools/releases/download/amber-modern-latest/AmberModernAndVtrEnabler.farconfig) a umiestnite súbor do priečinka Far Managera.
2. Úplne zatvorte všetky spustené inštancie Far Managera.
3. Spustite príkaz:

   ```bat
   far.exe /import AmberModernAndVtrEnabler.farconfig
   ```

4. Znovu spustite Far Manager.
