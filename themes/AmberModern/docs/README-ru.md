# 🎨 Цветовая тема Amber Modern

> Для тех, кто устал от классической синевы Far Manager и хочет современную, спокойную и приятную для глаз цветовую схему.

![Общий вид](demo.png)

## 📓 Примечания

1. Тема не изменяет правила раскраски файлов — используются уже настроенные в Far Manager правила.
2. Тема задаёт базовые цвета редактора и просмотрщика, но не включает собственную подсветку синтаксиса. Для редактора я использую [Monokai](https://github.com/joric/monokai) — рекомендую попробовать.

## 📦 Установка

Amber Modern использует расширенную цветовую палитру, поэтому требует включённого **Virtual Terminal Rendering**. Необходимая настройка уже включена в файл темы.

1. [Скачайте тему](https://github.com/LanKing/far-tools/releases/latest/download/AmberModernAndVtrEnabler.farconfig) и положите файл в каталог Far Manager.
2. Полностью закройте все запущенные экземпляры Far Manager.
3. Выполните в командной строке:

   ```bat
   far.exe /import AmberModernAndVtrEnabler.farconfig
   ```

4. Запустите Far Manager снова.
