# 🎨 Barevné téma Amber Modern

> Pro všechny, které unavuje klasický modrý vzhled Far Manageru a chtějí moderní, klidné a příjemné barevné schéma.

![Celkový vzhled](demo.png)

## 📓 Poznámky

1. Téma nemění pravidla zvýraznění souborů — používají se pravidla již nastavená ve Far Manageru.
2. Téma nastavuje základní barvy editoru a prohlížeče, ale neobsahuje vlastní zvýraznění syntaxe. Pro editor používám [Monokai](https://github.com/joric/monokai) a doporučuji jej vyzkoušet.

## 📦 Instalace

Amber Modern používá rozšířenou paletu barev, a proto vyžaduje zapnuté **Virtual Terminal Rendering**. Potřebné nastavení je již zapnuto v souboru tématu.

1. [Stáhněte téma](https://github.com/LanKing/far-tools/releases/download/amber-modern-latest/AmberModernAndVtrEnabler.farconfig) a umístěte soubor do adresáře Far Manageru.
2. Úplně zavřete všechny spuštěné instance Far Manageru.
3. Spusťte příkaz:

   ```bat
   far.exe /import AmberModernAndVtrEnabler.farconfig
   ```

4. Znovu spusťte Far Manager.
