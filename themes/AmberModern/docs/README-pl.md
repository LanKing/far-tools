# 🎨 Motyw kolorystyczny Amber Modern

> Dla osób zmęczonych klasycznym niebieskim wyglądem Far Managera, które szukają nowoczesnego, spokojnego i przyjaznego dla oczu schematu kolorów.

![Widok ogólny](demo.png)

## 📓 Uwagi

1. Motyw nie zmienia reguł kolorowania plików — używane są reguły już skonfigurowane w Far Managerze.
2. Motyw ustawia podstawowe kolory edytora i przeglądarki, ale nie zawiera własnego podświetlania składni. W edytorze używam [Monokai](https://github.com/joric/monokai) i polecam go wypróbować.

## 📦 Instalacja

Amber Modern korzysta z rozszerzonej palety kolorów, dlatego wymaga włączonego **Virtual Terminal Rendering**. Wymagane ustawienie jest już włączone w pliku motywu.

1. [Pobierz motyw](https://github.com/LanKing/far-tools/releases/download/amber-modern-latest/AmberModernAndVtrEnabler.farconfig) i umieść plik w katalogu Far Managera.
2. Całkowicie zamknij wszystkie uruchomione instancje Far Managera.
3. Uruchom polecenie:

   ```bat
   far.exe /import AmberModernAndVtrEnabler.farconfig
   ```

4. Ponownie uruchom Far Managera.
