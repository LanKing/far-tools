# 🎨 Колеравая тэма Amber Modern

> Для тых, каму надакучыў класічны сіні выгляд Far Manager і хто хоча сучасную, спакойную і прыемную для вачэй колеравую схему.

![Агульны выгляд](demo.png)

## 📓 Заўвагі

1. Тэма не змяняе правілы афарбоўкі файлаў — выкарыстоўваюцца правілы, ужо настроеныя ў Far Manager.
2. Тэма задае базавыя колеры рэдактара і праглядальніка, але не змяшчае ўласнай падсветкі сінтаксісу. Для рэдактара я выкарыстоўваю [Monokai](https://github.com/joric/monokai) і раю паспрабаваць.

## 📦 Усталяванне

Amber Modern выкарыстоўвае пашыраную колеравую палітру, таму патрабуе ўключанага **Virtual Terminal Rendering**. Патрэбная налада ўжо ўключана ў файл тэмы.

1. [Спампуйце тэму](../AmberModernAndVtrEnabler.farconfig) і змясціце яе ў каталог Far Manager.
2. Цалкам закрыйце ўсе запушчаныя асобнікі Far Manager.
3. Выканайце каманду:

   ```bat
   far.exe /import AmberModernAndVtrEnabler.farconfig
   ```

4. Зноў запусціце Far Manager.
