# 🎨 Кольорова тема Amber Modern

> Для тих, кому набрид класичний синій вигляд Far Manager і хто хоче сучасну, спокійну та приємну для очей кольорову схему.

![Загальний вигляд](demo.png)

## 📓 Примітки

1. Тема не змінює правила підсвічування файлів — використовуються правила, уже налаштовані у Far Manager.
2. Тема задає базові кольори редактора та переглядача, але не містить власного підсвічування синтаксису. Для редактора я використовую [Monokai](https://github.com/joric/monokai) — рекомендую спробувати.

## 📦 Встановлення

Amber Modern використовує розширену палітру кольорів, тому потребує ввімкненого **Virtual Terminal Rendering**. Необхідне налаштування вже ввімкнене у файлі теми.

1. [Завантажте тему](https://github.com/LanKing/far-tools/releases/download/amber-modern-latest/AmberModernAndVtrEnabler.farconfig) і помістіть файл у каталог Far Manager.
2. Повністю закрийте всі запущені екземпляри Far Manager.
3. Виконайте команду:

   ```bat
   far.exe /import AmberModernAndVtrEnabler.farconfig
   ```

4. Знову запустіть Far Manager.
