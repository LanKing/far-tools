# 🎨 Tema de color Amber Modern

> Para quienes están cansados del clásico aspecto azul de Far Manager y buscan una combinación de colores moderna, tranquila y agradable a la vista.

![Vista general](demo.png)

## 📓 Notas

1. El tema no modifica las reglas de coloreado de archivos: utiliza las reglas ya configuradas en Far Manager.
2. El tema establece los colores básicos del editor y del visor, pero no incluye resaltado de sintaxis propio. Para el editor utilizo [Monokai](https://github.com/joric/monokai) y recomiendo probarlo.

## 📦 Instalación

Amber Modern utiliza una paleta de colores ampliada, por lo que requiere tener activado **Virtual Terminal Rendering**. El ajuste necesario ya está habilitado en el archivo del tema.

1. [Descarga el tema](https://github.com/LanKing/far-tools/releases/download/amber-modern-latest/AmberModernAndVtrEnabler.farconfig) y coloca el archivo en el directorio de Far Manager.
2. Cierra por completo todas las instancias de Far Manager que estén en ejecución.
3. Ejecuta el comando:

   ```bat
   far.exe /import AmberModernAndVtrEnabler.farconfig
   ```

4. Inicia Far Manager de nuevo.
