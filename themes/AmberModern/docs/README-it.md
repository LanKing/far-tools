# 🎨 Tema colore Amber Modern

> Per chi è stanco del classico aspetto blu di Far Manager e desidera uno schema di colori moderno, sobrio e riposante per gli occhi.

![Panoramica](demo.png)

## 📓 Note

1. Il tema non modifica le regole di colorazione dei file: utilizza quelle già configurate in Far Manager.
2. Il tema imposta i colori di base dell'editor e del visualizzatore, ma non include un'evidenziazione della sintassi propria. Per l'editor utilizzo [Monokai](https://github.com/joric/monokai) e consiglio di provarlo.

## 📦 Installazione

Amber Modern utilizza una tavolozza di colori estesa e richiede quindi l'attivazione di **Virtual Terminal Rendering**. L'impostazione necessaria è già abilitata nel file del tema.

1. [Scarica il tema](https://github.com/LanKing/far-tools/releases/download/amber-modern-latest/AmberModernAndVtrEnabler.farconfig) e inserisci il file nella directory di Far Manager.
2. Chiudi completamente tutte le istanze di Far Manager in esecuzione.
3. Esegui il comando:

   ```bat
   far.exe /import AmberModernAndVtrEnabler.farconfig
   ```

4. Avvia nuovamente Far Manager.
